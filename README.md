# 📋 Fully Functional To-Do List Web Application

A **modern, responsive, persistent** task manager built with React + Vite.  
Create, edit, delete, filter, search, and save tasks automatically in your browser.

🔗 **Live Demo**: [https://functional-to-do-list-app.pages.dev/](#) 

---

## 🚀 Core Features

| Feature          | Description                                                       |
|------------------|-------------------------------------------------------------------|
| ➕ **Create**    | Add a task via the input form (press Enter or click Add).         |
| 👁️ **View**     | See all tasks displayed in a clean list.                         |
| ✏️ **Update**   | Edit task title inline (double‑click or click ✏️).               |
| ✅ **Complete**  | Toggle task status with a checkbox (strikethrough when done).    |
| 🗑️ **Delete**   | Remove a task with one click (🗑️).                              |
| 🔍 **Search**    | Filter tasks by title (case‑insensitive).                       |
| 🎚️ **Filter**   | Show **All** / **Completed** / **Pending** tasks.               |
| 💾 **Persist**   | Data saved automatically in Local Storage – survives refresh.    |
| 📱 **Responsive**| Works perfectly on mobile, tablet, and desktop.                  |

---

## 🛠️ Tech Stack

- **React** – UI components & state management  
- **Vite** – fast development & build tool  
- **CSS3** – custom properties (variables), flexbox, animations  
- **Local Storage API** – client‑side persistence  
- **Cloudflare Pages** – deployment & hosting
